## 📌 Overview

This project represents a modern agency website designed for showcasing:

- AI product design
- ML-based dashboard interfaces
- Data visualization UI
- SaaS product experiences
- Responsive web solutions

The layout is minimal, structured, and optimized for all devices.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/your-repo-name.git
cd your-repo-name
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
